Antes de qualquer linha de código, você precisa entender o que está acontecendo. Aqui não tem cópia e cola — tem compreensão.

O que é Python, afinal?

Python é uma linguagem de programação criada em 1991 por Guido van Rossum. Ela foi desenhada para ser legível como texto em inglês — diferente de linguagens como C ou Java, que exigem muito mais cerimônia.

Hoje Python é a linguagem dominante em inteligência artificial, análise de dados e automação. Quando você faz uma pergunta para uma IA, há uma boa chance de que Python esteja em algum ponto da cadeia.

💡

Analogia: Se programar é como dar instruções para alguém cozinhar, Python é como falar português simples. Outras linguagens são como ler um manual técnico em latim. Você chega no mesmo resultado, mas o caminho é bem diferente.

Variáveis — a base de tudo

Uma variável é uma caixinha com nome que guarda um valor. Simples assim. Em Python você não precisa declarar o tipo — a linguagem descobre sozinha.

# Guardando valores em variáveis
nome = "Ana"
idade = 28
altura = 1.65
ativo = True

# Usando as variáveis
print(f"Olá, {nome}! Você tem {idade} anos.")

Repare: não tem var, não tem let, não tem tipo declarado. Você escreve o nome, o sinal de igual, e o valor. Python cuida do resto.

str

String (texto)

Qualquer texto entre aspas. "olá" ou 'mundo'. Usado para nomes, mensagens, frases.

int / float

Números

int são inteiros (1, 42, -7). float têm casas decimais (3.14, 1.65). Python escolhe automaticamente.

bool

Booleano

Só dois valores possíveis: True ou False. Usado em condições e lógica.

list

Lista

Uma coleção ordenada de valores. [1, 2, 3] ou ["maçã", "banana"]. Pode misturar tipos.

Condições — tomando decisões

Todo programa precisa tomar decisões. Em Python, usamos if, elif e else para isso. A lógica é simples: "se isso for verdade, faça aquilo".

temperatura = 35

if temperatura > 30:
    print("Está muito quente! Beba água.")
elif temperatura > 20:
    print("Temperatura agradável.")
else:
    print("Está frio, pegue um casaco.")
⚠️

Atenção ao indentation! Python usa espaços (indentação) para definir blocos de código. Diferente de outras linguagens que usam chaves {}, aqui o espaço é obrigatório e significativo. Sempre use 4 espaços.

Funções — reutilizando código

Uma função é um bloco de código com nome, que você pode chamar quantas vezes quiser. Em vez de repetir as mesmas linhas, você define uma vez e chama sempre que precisar.

def calcular_imc(peso, altura):
    imc = peso / (altura ** 2)
    return round(imc, 2)

# Chamando a função com dados diferentes
print(calcular_imc(70, 1.75))  # → 22.86
print(calcular_imc(90, 1.80))  # → 27.78

A palavra def define a função. Os parênteses recebem os parâmetros (inputs). O return devolve o resultado. Simples.
